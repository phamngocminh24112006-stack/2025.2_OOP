8. Create Constructor method
Question: If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
- JAVA không cho phép xây dựng hai phương thức khởi tạo như vậy, vì cả category và title đều có kiểu dữ liệu string, nếu sử dụng thì JAVA sẽ không phân biệt được phải sử dụng phương thức nào để khởi tạo đối tượng, các phương thức khởi tạo phải khác nhau về signature ( Ví dụ: kiểu dữ liệu của tham số truyền vào phương thức hoặc số lượng tham số phải khác nhau)
